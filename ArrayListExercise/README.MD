ArrayList Utilities: Java Exercises
Μια συλλογή από βοηθητικές μεθόδους (utility methods) στη Java για την επεξεργασία λιστών τύπου ArrayList. Το project αυτό επικεντρώνεται σε βασικούς αλγορίθμους αναζήτησης και καταμέτρησης στοιχείων.

Περιγραφή
Ο κώδικας περιλαμβάνει υλοποιήσεις για:

Καταμέτρηση λέξεων: Υπολογισμός συχνότητας εμφάνισης του πρώτου στοιχείου μιας λίστας.

Εύρεση Min/Max: Εντοπισμός της μικρότερης και μεγαλύτερης τιμής σε μια λίστα ακεραίων (τόσο σε μία κοινή μέθοδο όσο και σε ξεχωριστές).

 Λειτουργίες
1. wordCount(ArrayList<String> list)
Αυτή η μέθοδος παίρνει μια λίστα από Strings και μετράει πόσες φορές εμφανίζεται η πρώτη λέξη της λίστας μέσα σε όλη τη συλλογή.

Input: ["java", "python", "java", "c"]

Output: 2 (για τη λέξη "java")

2. minMax(ArrayList<Integer> list)
Μια "all-in-one" μέθοδος που διατρέχει τη λίστα μία φορά και εκτυπώνει ταυτόχρονα το ελάχιστο και το μέγιστο στοιχείο.

3. min & max (Ξεχωριστές Μέθοδοι)
Αυτοτελείς συναρτήσεις που επιστρέφουν την ελάχιστη ή μέγιστη τιμή μιας λίστας ακεραίων για χρήση σε υπολογισμούς.

⚙️ Τεχνολογίες & Εργαλεία
